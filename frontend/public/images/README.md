# Imagens da Mesa de Poker

Coloque aqui a imagem da mesa de poker.

## Nome do arquivo:

- `poker-table.jpg` (preferido)
- `poker-table.png`
- `poker-table.webp`

O sistema tenta carregar nesta ordem: JPG → PNG → WEBP

## Especificações recomendadas:

- **Formato:** JPG (melhor compressão) ou PNG
- **Tamanho:** 1920x1080 ou maior (ideal: 2560x1440)
- **Peso:** < 2MB para melhor performance
- **Aspecto:** Mesa de poker vista de cima

## Onde encontrar imagens gratuitas:

- **Unsplash:** https://unsplash.com/s/photos/poker-table
- **Pexels:** https://www.pexels.com/search/poker%20table/
- **Pixabay:** https://pixabay.com/images/search/poker%20table/

## Como adicionar:

1. Baixe uma imagem de mesa de poker
2. Renomeie para `poker-table.jpg` (ou .png/.webp)
3. Coloque nesta pasta: `frontend/public/images/`
4. Recarregue a página

## Exemplo de URL para baixar:

- Unsplash tem várias imagens gratuitas de mesas de poker
- Procure por "poker table top view" ou "poker table green felt"
